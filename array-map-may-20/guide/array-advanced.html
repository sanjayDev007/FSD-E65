Iterating Over Arrays üîÑ

forEach()

What it does: Runs a function for each item in the array.

Analogy: Like checking every item in your shopping cart, one by one.

let fruits = ['üçé', 'üçå', 'üçí']; fruits.forEach(function(fruit, index) { console.log(index, fruit); }); // 0 'üçé' // 1 'üçå' // 2 'üçí'

map()

What it does: Creates a new array by transforming each item.

Analogy: Like putting stickers on every item in your cart and making a new cart with the stickers.

let prices = [10, 20, 30]; let doubled = prices.map(price => price * 2); // doubled is [20, 40, 60]

filter()

What it does: Creates a new array with only the items that pass a test.

Analogy: Like picking only the ripe fruits from your cart.

let numbers = [1, 2, 3, 4, 5]; let even = numbers.filter(n => n % 2 === 0); // even is [2, 4]

reduce()

What it does: Reduces the array to a single value by combining items.

Analogy: Like adding up the total price of everything in your cart.

let prices = [10, 20, 30]; let total = prices.reduce((sum, price) => sum + price, 0); // total is 60

Searching in Arrays üîç

find()

What it does: Returns the first item that matches a test.

let users = [ { name: 'Alex', age: 15 }, { name: 'Sam', age: 20 } ]; let teen = users.find(u => u.age < 18); // teen is { name: 'Alex', age: 15 }

includes()

What it does: Checks if the array contains a value.

let fruits = ['apple', 'banana', 'cherry']; fruits.includes('banana'); // true fruits.includes('grape'); // false

indexOf()

What it does: Returns the index of a value, or -1 if not found.

let fruits = ['apple', 'banana', 'cherry']; fruits.indexOf('banana'); // 1 fruits.indexOf('grape'); // -1

Multidimensional Arrays üßÆ

What are they?

Arrays can hold other arrays! This is useful for things like grids, tables, or matrices.

let grid = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]; // Access the number 5: let middle = grid[1][1]; // 5

Summary & Practice üìù

  • forEach: Do something for every item
  • map: Make a new array by transforming items
  • filter: Make a new array with only some items
  • reduce: Turn an array into a single value
  • find/includes/indexOf: Search for items
  • Multidimensional arrays: Arrays inside arrays!
