String Methods for Arrays ğŸ”¤

Arrays have some handy methods to turn them into strings or combine them.

join()

What it does: Combines all array elements into a single string, separated by what you choose.

let fruits = ['apple', 'banana', 'cherry']; let text = fruits.join(', '); // text is 'apple, banana, cherry'

toString()

What it does: Converts the array to a comma-separated string.

let fruits = ['apple', 'banana', 'cherry']; let text = fruits.toString(); // text is 'apple,banana,cherry'
