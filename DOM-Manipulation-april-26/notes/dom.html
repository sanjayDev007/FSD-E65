What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as a tree of nodes where each node represents a part of the document.

How the DOM Works

When a web page is loaded, the browser creates a Document Object Model of the page. The DOM is constructed as a tree of objects:

  • Document Node: The top-level node representing the entire document.
  • Element Nodes: Represent HTML elements (like HTML, HEAD, BODY, DIV, P, etc.).
  • Attribute Nodes: Represent attributes of elements (like class, id, style, etc.).
  • Text Nodes: Contain the text inside elements.
  • Comment Nodes: Represent HTML comments (not shown in visualization).

The DOM provides a way for JavaScript to access and manipulate the content, structure, and style of a document. When JavaScript makes changes to the DOM, the browser updates the rendered page accordingly.

JavaScript Interaction with DOM

JavaScript can:

  • Add, change, and remove HTML elements and attributes
  • Change CSS styles
  • React to existing events and create new ones
  • Create and manipulate event handlers

DOM Tree Traversal

JavaScript can navigate the DOM tree using properties like:

  • parentNode: Access the parent of a node
  • childNodes: Access all children of a node
  • firstChild, lastChild: Access first or last child
  • nextSibling, previousSibling: Access adjacent nodes
